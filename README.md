🥟 Sistema de Pedidos para Pastelaria

Projeto desenvolvido para fins de estudo, inspirado em uma necessidade real de uma pastelaria local da cidade onde resido.

Este projeto consiste em um sistema de pedidos online para pastelaria, com foco em simplicidade, organização e boa experiência de uso.

A ideia inicial foi criar um sistema que permitisse aos clientes visualizar produtos, adicionar itens ao carrinho e realizar pedidos de forma prática.

⚠️ Importante:
O sistema ainda está em desenvolvimento. Algumas funcionalidades ainda estão sendo implementadas e outras passarão por melhorias e correções.

🎯 Objetivo do Projeto

O projeto foi pensado inicialmente para ajudar uma pastelaria local, servindo também como prática de desenvolvimento full-stack, envolvendo front-end, back-end e banco de dados.
